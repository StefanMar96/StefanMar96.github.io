Project description

The project consists of the following:

1. Come up with a suggestion for a project
2. Devide the responibility and choose a time for each milestone
3. Decide on a licence for the project
4. Make a one page product description
5. Make a 2 minute introduction video for the project
6. Make a slideshow presentation for the project

Project suggestion

We wanted to make something similar to the "BRIO labyrinth" but add our own twist to it. We decided to have the size meter*meter and it is supposed to be used for team-building and for fun. We also wanted to make a ball to be used in the labyrinth, but unfortionately we could not finish that item due to covid-19.
