Making a website

The purpose of this project was to design a functional website that will contain my resume, as well as all the projects that i complete in the course Manufacturing Processes (VÉL403G) at the University of Iceland.

The beginning

The first thing that I needed to do was to educate myself on the basics in Html programming. I decided to view the material that the professor of the course had uploaded to Youtube, as well as the website w3schools. All of the information I gathered proved to be extremely helpful as the project became much clearer the more I learned about the Html language.

Since I had no previous experience with HTML, I decided to use a template from HTML5UP to use as a starting point. The template that I started with is called "Massively", and I picked that template because of its looks.

Programs

Brackets

For this project, I downloaded Brackets to edit the HTML code which I got from HTML5UP. The reason for why I chose to use Brackets is because you can click on the lightning in the top-right corner to see any changes you make in the code as soon as you save the changes, which is a great feature.

Github

I already had a Github account, so I just needed to create a repository to host the files which make up the website. I also used Github to host my website duo to the fact that it is extremely easy to do it from there.

I chose a convenient name for my repository, which is the same as the link I want people to be able to access my website through.

Git

Lastly, I had to download the program Git, which is used to track changes in a source code during software development. I mainly used Git to push the changes I made in the source code to my repository on Github.

I went to the folder where all the files for my website were, and I pressed the right click buttun on the mouse and then pressed "Git bash here"

Then i typed the following into the terminal:

1. echo "# StefanMar96.github.io" >> README.md

2. git init

3. git add README.md

4. git commin -m "first commit"

5. git branch -M main

6. git remote add origin https://github.com/StefanMar96/StefanMar96.github.io.git

7. git push -u origin main

8. git add .

9. git commit -m "first commit" .

10. git push

Now the files will have been pushed to my repository and it should look like this.

Now the only thing left to do is, click on settings on the repository, scroll down to GitHub Pages and click on source and change it to Main. Then I can see the url which can be used to access the website.
